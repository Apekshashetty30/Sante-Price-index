Sante-Price Index
Project Overview
Sante-Price Index is an Android-based mobile application developed to support small-scale vendors in weekly markets (Santes). The application helps vendors calculate fair selling prices for their products by considering daily mandi prices, transportation charges, and additional expenses. The system aims to improve pricing decisions, increase profit opportunities for vendors, and provide fair pricing for customers.

 Objectives
- To help vendors check current mandi prices easily
- To calculate accurate product selling prices
- To reduce manual calculation errors
- To support small vendors with smart pricing assistance
- To create a simple and user-friendly Android application

 Features
- Vendor Login System
- Daily Mandi Price Management
- Product Price Calculation
- Transport and Additional Expense Calculation
- Final Selling Price Suggestion
- Simple and Easy-to-Use Interface
- Vendor Dashboard

 Technologies Used
- Android Studio
-  Kotlin
- XML
- Firebase
- Git & GitHub
